# Disable Conversation Mode

Turn off auto-speak. Claude responses will stop being spoken automatically.

## Instructions

1. Run: `rm -f $CLAUDESPEAK_HOME/conversation-mode`
2. Run: `claudespeak --async "Conversation mode off"` (final spoken confirmation)
3. Report: "Conversation mode: **off**. Auto-speak disabled. Use `/voice-on` to re-enable."
